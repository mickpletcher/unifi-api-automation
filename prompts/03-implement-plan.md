Implement a feature from plan and track work.

Steps:
1. Read `.github/specs/<feature-slug>/spec.md` and `plan.md`.
2. Create or update `.github/specs/<feature-slug>/tasks.md` using `.github/specs/templates/tasks-template.md`.
3. Execute planned code changes in small commits.
4. Update `tasks.md` status as work progresses.
5. Run all validation commands from the plan.
6. Summarize results and remaining work.

Output:
1. List completed tasks.
2. List changed files.
3. Report validation results.
4. List remaining tasks.
